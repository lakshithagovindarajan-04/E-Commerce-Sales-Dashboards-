# E‑Commerce Sales Dashboards | SQL & Power BI 

## 📌 Overview
Developed interactive dashboards to analyze Amazon & Flipkart sales data, providing insights into revenue, profit, order volume, and customer segments.

## ⚙️ Tech Stack
- SQL (joins, aggregations, date functions) for data cleaning & transformation
- Power BI (DAX measures, dynamic filters, charts, maps) for visualization

## 📊 Features
- Cleaned & transformed 25k+ sales records ensuring data integrity
- Built KPIs: Year‑over‑Year growth, profit margin, average order value
- Identified top 5 product categories driving 60% of profit
- Highlighted underperforming regions to support resource allocation decisions

## 📝 How to Run
1. Download `Ecommerce_Sales_Dashboard.pbix`
2. Open in Microsoft Power BI Desktop
3. Use filters & slicers to explore sales KPIs
